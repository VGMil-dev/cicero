# Introducción a Cicero

Bienvenido a la documentación técnica de **Cicero**, el asistente inteligente para mejorar tus habilidades de oratoria.

## 🎯 El Problema
Muchos oradores, tanto principiantes como experimentados, sufren del uso excesivo de **muletillas** (*fillers*) como "ehh", "este", "bueno" o "digamos". Estas palabras restan profesionalismo y claridad al mensaje.

## 💡 Nuestra Solución
Cicero ofrece una aplicación web progresiva (PWA) sencilla donde puedes:
1.  **Grabar** tus discursos o ensayos de forma local.
2.  **Analizar** el audio automáticamente para detectar muletillas.
3.  **Visualizar** métricas clave sobre la fluidez y limpieza de tu oratoria.

Esta documentación está organizada para ayudarte a entender las decisiones técnicas y la estructura del proyecto:

- **[Arquitectura](./arquitectura/):** Explica el diseño del sistema, el stack tecnológico y los patrones utilizados (Hexagonal, Vertical Slicing).
- **Guías de Desarrollo:** (Próximamente) Instrucciones para configurar el monorepo y contribuir al código.

Empieza explorando la sección de **Arquitectura** para conocer el corazón técnico de Cicero.
